# 📍 Google Maps Lead Scraper

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![Playwright](https://img.shields.io/badge/playwright-v1.42.0-green)
![License](https://img.shields.io/badge/license-MIT-orange)

> **A professional-grade, asynchronous lead generation tool.** This scraper leverages **Playwright Stealth** to extract business intelligence (Names, Phone Numbers, and Addresses) from Google Maps with human-like behavior.

---

## 🚀 Key Features

* ⚡ **Asynchronous Power:** Built with `asyncio` to scrape multiple locations (like Tongi and Uttara) simultaneously.
* 🕵️ **Stealth Integration:** Uses `playwright-stealth` to bypass bot detection and minimize IP flagging.
* 📊 **Deep Extraction:** Automatically clicks each listing to extract "hidden" data not available on the summary page.
* 📂 **Ready-to-Use Data:** Generates a clean, **UTF-8-SIG CSV** file compatible with Excel and Google Sheets.

---

## 🛠️ Prerequisites

| Requirement | Description |
